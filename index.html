Research

I'm interested in 3D computer vision, deep learning, optimization. Much of my research is about single view 3D recontruction from point clouds, 3D implicit surface reconstruction. Representative papers are highlighted.

GridPull: Towards Scalability in Learning Implicit Representations from 3D Point Clouds.
Chao Chen, Yu-Shen Liu*, Zhizhong Han,

ICCV, 2023

We propose GridPull to reconstruct surface fastly from large scale point clouds without using neural networks.

Unsupervised Inference of Signed Distance Functions from Single Sparse Point Clouds without Learning Priors.
Chao Chen, Yu-Shen Liu*, Zhizhong Han,

CVPR, 2023
github / open access / arXiv

We introduce a neural network to infer SDFs from single sparse point clouds without using signed distance supervision, learned priors or even normals.

Latent Partition Implicit with Surface Codes for 3D Representation
Chao Chen, Yu-Shen Liu*, Zhizhong Han,

ECCV, 2022
project page / arXiv

We introduce a novel implicit representation to represent a single 3D shape as a set of parts in the latent space, towards both highly accurate and plausibly interpretable shape modeling.

Unsupervised Learning of Fine Structure Generation for 3D Point Clouds by 2D Projections Matching
Chao Chen, Yu-Shen Liu*, Zhizhong Han,
Matthias Zwicker,
ICCV, 2021
github / open access / arXiv / media report

We introduce a method to enable the unsupervised learning of 3D point cloud generation with fine structures by 2D projection matching.

DRWR: A Differentiable Renderer without Rendering for Unsupervised 3D Structure Learning from Silhouette Images
Zhizhong Han, Chao Chen, Yu-Shen Liu*, Matthias Zwicker,
ICML, 2020
github / PMLR / arXiv

We propose a Differentiable Renderer Without Rendering (DRWR) for unsupervised 3D point cloud reconstruction from 2D silhouette images.

ShapeCaptioner: Generative Caption Network for 3D Shapes by Learning a Mapping from Parts Detected in Multiple Views to Sentences
Zhizhong Han, Chao Chen, Yu-Shen Liu*, Matthias Zwicker,
ACM MM, 2020
DOI / arXiv

We propose ShapeCaptioner to enable 3D shape captioning from semantic parts detected in multi-views.
